# Swagger Petstore API Automation Framework
This project is a REST API Automation Framework built using Java, Rest Assured, TestNG, and Maven for Swagger Petstore API.

Modules Covered:
- User Module
- Pet Module

Features:
- CRUD operations validation
- Request & Response logging
- POJO classes
- TestNG assertions
- Data-driven testing
- Reusable utility methods
- Extent Reports
- Maven build support
